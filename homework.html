HW0

Optional Data Camp assignments.

Register for our course on Data Camp at this link, and complete the assignment there: https://www.datacamp.com/groups/shared_links/e05929250408228fa302681f53f66bdc48e8142d

If you are already comfortable with R, you don’t need to do this. If you would like a review, you are welcome to do this. If you haven’t seen R before, you should treat this as a required assignment even though it won’t be graded. If you have not used R much/at all yet, this assignment may feel overwhelming. Don’t worry, we will talk about this material in more depth in future classes. This is a first pass.
